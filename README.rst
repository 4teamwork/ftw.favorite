ftw.favorite
============

This Package provides a Content-Type like Favorite like in Plone 3.


Usage
-----

- Add ``ftw.favorite`` to your buildout configuration:

::

    [instance]
    eggs +=
        ftw.favorite

- Install the generic setup profile of the package
